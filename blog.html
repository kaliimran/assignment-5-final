1.What is DOM and what is the purpose of DOM?

Ans: The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.One of the main reasons for developing the DOM is to maintain a standard programming interface for HTML documents, which is language-independent.

2.How will you select HTML elements using DOM? Name the DOM methods

There are elements in a DOM using selectors:

  • getElementById(): search element by element_id.
  • getElementsByTagName(): search element by tag name
  • getElementsByClassName(): search element by class name.
  • getElementsByName(): search element by name attribute.

3.What is event bubble?

Ans: Event Bubble is a type of DOM event propagation where the event first triggers on the innermost target element, and then successively triggers on the ancestors (parents) of the target element in the same nesting hierarchy till it reaches the outermost DOM element or document object.

4.What is a callback function and why will you use it?

Ans: A JavaScript callback is a function which is to be executed after another function has finished execution.A more formal definition would be - Any function that is passed as an argument to another function so that it can be executed in that other function is called as a callback function.
